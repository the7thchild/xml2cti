# xml2cti
Playstation 1 (PSX) A program to convert XML file generated by dumpsxiso to CTI (BuildCD)

CTI file is an obsolete format as buildcd is no longer working on recent version of windows.
Thanks to John "Lameguy" Wilbert Villamor/Lameguy64, Silent (CookiePLMonster) and spicyjpeg (https://github.com/Lameguy64/mkpsxiso). A modern and even better tool is created for PSX ISO management.
I have created this program so that the xml file generated by dumpsxiso can be converted CTI format, which is very useful for SNSYS CD Emulator owners.

Usage: xml2cti input.xml output.cti
